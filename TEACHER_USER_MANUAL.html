  • 1. Wait and try again (server may be busy)

  • 2. Simplify topic description

  • 3. Check all required fields are filled

  • 4. Use more specific vocabulary suggestions

  • 5. Try different topic if problem continues


    • #### Issue: Students Can't Access Module
    Possible Causes:
  • Module not published

  • Level restrictions active

  • Technical problems


    • Solutions:
  • 1. Verify module is published and active

  • 2. Check level access settings

  • 3. Test module yourself to confirm functionality

  • 4. Review student enrollment status

  • 5. Check for system-wide issues

    Symptoms:
  • High dropout rate

  • Low completion scores

  • Negative feedback


    • Solutions:
  • 1. Review CEFR level alignment

  • 2. Add more scaffolding and support

  • 3. Include prerequisite review

  • 4. Simplify complex exercises

  • 5. Add hint systems

    Symptoms:
  • Very short completion times

  • High scores but low engagement

  • Feedback that module is too easy


    • Solutions:
  • 1. Add more challenging exercises

  • 2. Include extension activities

  • 3. Increase content depth

  • 4. Add critical thinking elements

  • 5. Consider raising CEFR level


    • #### Issue: AI Tutor Responses Are Poor
    Symptoms:
  • Students complain about AI interactions

  • Conversations don't flow well

  • Inappropriate or unhelpful responses


    • Solutions:
  • 1. Refine AI personality description

  • 2. Be more specific about focus areas

  • 3. Add more common mistakes examples

  • 4. Test conversation scenarios thoroughly

  • 5. Provide more cultural context
