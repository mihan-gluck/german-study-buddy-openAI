# AI-Assisted Module Creation Guide

## Overview
The AI-assisted module creation system allows teachers to generate complete learning modules using artificial intelligence. This feature significantly reduces the time needed to create comprehensive educational content.

## How to Use

### 1. Access the Feature
1. Login as a **TEACHER** or **ADMIN**
2. Navigate to **Learning Modules** page
3. Click **"Create New Module"** button
4. Choose **"Create with AI Assistant"** option

### 2. Provide Basic Information
Fill out the AI generation form with:

#### Required Fields:
- **Target Language**: The language students will learn (English/German)
- **Native Language**: Student's native language (English/Tamil/Sinhala)
- **Level**: CEFR level (A1, A2, B1, B2, C1, C2)
- **Category**: Type of content (Grammar, Vocabulary, Conversation, etc.)
- **Difficulty**: Learning difficulty (Beginner, Intermediate, Advanced)
- **Description**: Detailed description of what the module should teach

#### Optional Fields:
- **Duration**: Estimated completion time (10-120 minutes)
- **Module Type**: Standard or Role-play module

### 3. AI Generation Process
1. Click **"Generate Module with AI"**
2. AI analyzes your requirements
3. Generates comprehensive module content:
   - 15-25 vocabulary words with translations
   - 3-5 learning objectives
   - 5-8 practice exercises
   - Conversation examples
   - Cultural context (if requested)
   - AI tutor configuration

### 4. Review and Customize
After generation, you can:
- **Save Module**: Use as-is
- **Edit & Customize**: Modify content before saving
- **Start Over**: Generate a new module

**Poor Description:**

## Tips for Best Results

### 1. Be Specific
- Include specific scenarios and contexts
- Mention key vocabulary areas
- Describe the learning outcomes

### 2. Consider Your Students
- Match difficulty to student level
- Include cultural context for better understanding
- Focus on practical, real-world usage

### 3. Use the Right Level
- **A1-A2**: Basic vocabulary, simple sentences
- **B1-B2**: Complex conversations, grammar structures
- **C1-C2**: Advanced topics, nuanced language

- Fallback system for API failures

## Troubleshooting

### Common Issues:

**"Generation Failed"**
- Check internet connection
- Try again with simpler description
- Contact administrator if persistent

**"Authentication Error"**
- Ensure you're logged in as TEACHER/ADMIN
- Refresh the page and try again

**"Invalid Response"**
- The AI occasionally returns malformed data
- System will retry automatically
- Fallback module will be created if needed

### Getting Help:
- Contact system administrator
- Check the console for error messages
- Try the manual module creation as alternative
