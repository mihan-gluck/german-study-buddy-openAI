# Teacher Module Testing Guide

## Overview
Teachers can now test their own modules to experience exactly what students will encounter during AI tutoring sessions. This feature allows teachers to quality-check their modules and understand the student learning experience.

## How to Test Your Modules

### 1. Access the Test Feature
1. **Login as a TEACHER** (e.g., TEA001/password123)
2. **Navigate to Learning Modules** page
3. **Find your module** in the list
4. **Click the "Test Module" button** (green button with play icon)

### 2. Confirmation Dialog
When you click "Test Module", you'll see a confirmation dialog explaining:
- What the test will do
- Features you'll experience
- That it's a test session (won't affect student data)

## Troubleshooting

### ğŸ”§ **Common Issues**

**"Test Module button not visible"**
- Ensure you're logged in as TEACHER/ADMIN
- Check that you created the module (teachers can only test their own)
- Refresh the page if needed

**"Session won't start"**
- Check internet connection
- Verify module is active
- Try refreshing and starting again

**"AI not responding"**
- Check backend server status
- Verify OpenAI API configuration
- Try a different browser if issues persist

**"Voice features not working"**
- Check microphone permissions
- Test with Audio Test feature first
- Ensure browser supports speech recognition
